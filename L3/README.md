# L3 testing template

## Install 
### install python lib
```
pip install -r requirements.txt
```

Add the chromedriver to your PATH

## Run
```
robot ./test
```

### Architecture
```
└───L3
    ├───config
    ├───keywords
    ├───page_object
    ├───test
    │   └───index
    ├───utility
        └───webdriver
```
